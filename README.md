<h1 align="center"><a href="https://github.com/SSM-Demo" target="_blank">SSM-Demo</a></h1>

> 一个最简单的 SpringMVC + Spring + Mybatis 框架整合 Demo，在此基础上增加一些功能，比如分页等

------------------------------

## 说明

- 作者：Waciao

- 环境：

    - JDK        1.8.0_211
    - Maven      3.6.3
    - Mysql      8.0.19
    - Tomcat     9
    - Pagehelper 5.1.11
    - IntelliJ IDEA
    - SSM
        - Spring           5.2.4
        - mybatis          3.5.4
        - spring-mybatis   2.0.4
